# Project «Sedona»
[![Build status][travis-image]][travis-url] [![Dependency status][dependency-image]][dependency-url]


### Install
```
npm install
```

### Run
```
npm run start
```

### Build and run with Docker
```
sudo docker build -t sedona .
docker run -p 5000:80 --name sedona -d sedona
```
